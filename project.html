Технологический стек проекта

Проект "BrowHarmony" реализован с использованием современных технологий и лучших практик веб-разработки. Мы создали масштабируемую и производительную платформу для успешного ведения бьюти-бизнеса.

Frontend-разработка

Интерфейс пользователя создан с использованием современных веб-технологий для обеспечения высокой производительности, отзывчивости и премиального пользовательского опыта.

HTML5 CSS3 & SASS JavaScript (ES6+) Vue.js 3 Bootstrap 5 jQuery

Адаптивная верстка, PWA-технологии, оптимизация под мобильные устройства

Backend-разработка

Серверная часть построена на надежной и производительной платформе с использованием WordPress как основной CMS с кастомными плагинами для расширенной функциональности.

WordPress 6.0+ PHP 8.1+ REST API WooCommerce Custom Plugins

Многоуровневая архитектура, API-шлюз для интеграций, кэширование

Архитектура базы данных

Для хранения и обработки данных используется высокопроизводительная система управления базами данных с оптимизированной структурой и репликацией для обеспечения отказоустойчивости.

Пользователи
• ID пользователя (primary key)
• Имя и контактные данные
• История посещений
• Баланс лояльности

• ID услуги и мастера
• Дата и время визита
• Статус записи
• Сумма и способ оплаты
Услуги
• Категория услуги
• Название и описание
• Цена и длительность
• Фото результата
• Рейтинг и отзывы
Транзакции
• ID транзакции
• Сумма и валюта
• Статус платежа
• Дата и время
• Данные электронного чека

Google Tag Manager

Оптимизация и производительность

Для обеспечения максимальной производительности сайта были применены следующие оптимизации:

Кэширование

Redis для сессий, Varnish для статики, CDN для медиафайлов

Сжатие

GZIP/Brotli для ресурсов, WebP для изображений, минификация кода

База данных

Индексы, оптимизированные запросы, репликация чтения

DevOps и инфраструктура

Проект построен с использованием современных DevOps практик:

Docker GitHub Actions CI/CD AWS/GCP Cloud Nginx + PHP-FPM Let's Encrypt SSL Sentry (мониторинг ошибок)

Автоматическое развертывание, мониторинг 24/7, резервное копирование, защита от DDoS
